# Changes from RTP Model Run
## This model is Housing Forward

DESCRIPTION: An alternative that would focus on achieving housing need through the preservation and conversion of existing housing to workforce housing. This alternative would include a requirement that a defined percentage of a jurisdiction's housing stock is owned or rented by local workers. The alternative would adopt only a subset of the proposed policy components (shown below), without zoning, bonus unit, coverage or most fee incentives. This alternative would not convert unused development rights to bonus units, would not include coverage incentives for multifamily or changes to the Excess Coverage Mitigation Fee, and would not provide new incentives for Junior ADUs, or for "missing middle" housing. There would be no mapping changes from the "No Project" alternative.
